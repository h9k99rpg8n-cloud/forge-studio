# Forge Studio

Forge Studio es un editor 3D web enfocado en modelado por bloques desde navegador y dispositivos móviles.

## Estado actual

**Alfa 0.1**

Funciones incluidas:

- Escena 3D con Three.js
- Cámara orbital táctil
- Zoom con dos dedos
- Cuadrícula 3D
- Cubo inicial
- Agregar cubo
- Borrar cubo
- Selección básica con doble toque
- Subir y bajar cubo
- Cambiar color del cubo
- Interfaz móvil inicial

## Publicación con GitHub Pages

Configura el repositorio en:

```text
Settings → Pages → Deploy from a branch → main → /root
```

El enlace esperado será:

```text
https://h9k99rpg8n-cloud.github.io/forge-studio/
```

## Objetivo

Forge Studio busca convertirse en una herramienta web para crear modelos 3D por bloques, pensada para móviles, navegador y proyectos de creación de assets.
